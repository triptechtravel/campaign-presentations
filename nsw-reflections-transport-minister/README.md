# NSW Passenger Movement Analysis — Transport Minister Request

Analysis of CamperMate app location data for NSW, comparing Jan–Mar 2025 vs Jan–Mar 2026. Prepared for Destination NSW / Transport Minister briefing.

**Not a campaign** — this is an ad-hoc data request using CamperMate's foreground location pings to measure travel distance and multi-region movement in NSW.

## Key Findings

- **Domestic travellers** are covering significantly less distance YoY (median spread 46.7 → 8.5 km in March, -82%)
- **International travellers** are also down but still touring (median 255.5 → 143.5 km, -44%)
- **Single-region domestic users** rose from 59.7% → 71.6% in March
- An "Unclassified" cohort (~2,335 users in Mar 2026) shows near-zero movement — likely domestic users casually browsing the app from home

## Methodology

- **Source**: `analytics_prod.stg_geozone_user_event` (partitioned by `utc_created_timestamp`, clustered by `event_name, state, country, tourism_region`)
- **Events**: `location_change` + `location_ping` for all apps; all events for CamperMate 5.x iOS (which embeds user GPS in every event via `$.userAgent` meta)
- **Travel spread**: ST_DISTANCE between min/max lat-lng bounding box per user per month
- **Region assignment**: ST_CONTAINS spatial join against `shape_tourism_region` (Destination NSW polygons)
- **Home country**: `stg_geozone_user.home_country` for matched users; locale extracted from `$.userAgent` in raw events for unmatched CamperMate 5.x users (sampled every 3rd day across 6 months, materialized to `dbt_hg.tmp_user_locale`)
- **Minimum threshold**: 3+ pings per user per month

## Data Caveats

1. CamperMate users skew toward camping/touring travellers (caravanners, road-trippers)
2. We know home **country** (via device locale) but not home **state** — cannot isolate intra-NSW locals vs interstate visitors
3. The `stg_geozone_user` dbt model only handles CamperMate app versions 5.0.0–5.2.0; users on 5.3+ require locale extraction from raw event data
4. thl Roadtrip uses independent versioning (5.x ≠ new React Native app) — treated as old native app, only `location_change` events used
5. The Unclassified cohort are CamperMate 5.x users not captured by the 58-day locale sample — their near-zero travel spread suggests domestic/local users
